What is Cloud Computing?

Cloud computing is on-demand availability of computing resources (servers, storage, databases) delivered over the internet.

Core AWS Services

ServiceDescription
EC2Virtual servers in the cloud
S3Object storage for files
RDSManaged databases
VPCVirtual private cloud
IAMIdentity and access management

AWS CLI Basics

aws ec2 describe-instances
aws ec2 run-instances --image-id ami-xxx --count 1 --instance-type t2.micro
