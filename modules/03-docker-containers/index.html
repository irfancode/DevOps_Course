What are Containers?

Containers are lightweight, portable units that package an application with all its dependencies. They are faster and more efficient than virtual machines.

Key Terms

TermDescription
ImageA read-only template for creating containers
ContainerA running instance of an image
DockerfileScript to build an image
RegistryStorage for images (Docker Hub)
VolumePersistent data storage

Essential Docker Commands

docker images                    # List local images
docker pull nginx:latest    # Download image
docker build -t myapp .     # Build from Dockerfile
docker run -d myapp      # Run container in background
docker ps                # Running containers
docker logs container     # View logs
docker exec -it container bash  # Enter container
docker stop container     # Stop container
docker rm container       # Remove container

Basic Dockerfile

FROM node:18-alpine
WORKDIR /app
COPY package*.json ./
RUN npm ci --only=production
COPY . .
EXPOSE 3000
ENV NODE_ENV=production
CMD ["node", "server.js"]
