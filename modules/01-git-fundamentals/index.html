What is Version Control?

Version Control is a system that records changes to files over time so you can recall specific versions later. Think of it as a time machine for your code!

Git vs GitHub

TermWhat it is
GitA version control system (the software)
GitHubA cloud platform that hosts Git repositories

Essential Git Commands

# Initialize repository
git init

# Check status
git status

# Add files
git add .
git add filename.txt

# Commit changes
git commit -m "Your message"

# View history
git log
git log --oneline

Branching

# Create branch
git branch feature-name

# Switch to branch
git checkout feature-name
git switch feature-name

# Create and switch
git checkout -b feature-name

# Merge branch
git checkout main
git merge feature-name
